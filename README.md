📚 PYQ Practice

«Practice smarter. Revise better. Perform with confidence.»

PYQ Practice is a modern platform designed to help students prepare effectively through Previous Year Questions (PYQs). By organizing questions into focused practice sessions and revision-friendly formats, the platform makes exam preparation more structured, efficient, and engaging.

Whether you're preparing for school exams or competitive examinations, PYQ Practice helps you strengthen concepts, improve speed and accuracy, and build confidence through consistent practice.

---

✨ Features

- 📖 Organized Previous Year Question (PYQ) collections
- 🎯 Topic-wise practice sessions
- ⚡ Quick revision tools
- 📊 Progress tracking
- ⏱️ Timed practice for better speed and accuracy
- 🔍 Easy navigation and question discovery
- 📱 Responsive design for desktop and mobile
- 🚀 Clean, intuitive, and distraction-free interface

---

🎯 Why PYQ Practice?

Traditional question banks and PDFs often make revision repetitive and difficult to organize. PYQ Practice provides a streamlined learning experience by helping students focus on relevant questions, monitor their progress, and revise efficiently—all in one place.

---

🛠 Tech Stack

- Language: TypeScript
- Runtime: Node.js
- Frontend: Web-based application
- Styling: HTML & CSS

---

🚀 Getting Started

Prerequisites

- Node.js (LTS version recommended)
- npm or another compatible package manager

Installation

git clone <repository-url>
cd PYQ-Practice
npm install

Run the Project

npm run dev

or

npm start
